## Name
* IWASAKI Kai

## Overview
* Perfect number in C++
* C++による完全数の判定と個数を求めるプログラム

## Description
* このプログラムは、整数 N が、完全数か判定する。
* 整数 N が、完全数であれば、個数を出力する。
* ただし、N は 1 以上 10＾12  以下の整数とする。
