## Overview
* Perfect number in C++
* C++による完全数の判定と個数を求めるプログラム

## Description
* このプログラムは、完全数の判定と個数を求める。
* (i). 入力を1つの整数(Nとする)にし、1 以上 N 以下の範囲で考える場合
* (ii). 入力を2つの整数(N,Mとする)にし、N 以上 M 以下の範囲で考える場合
